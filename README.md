# notification-service

## Description

Notification mcroservice developed during Rocketseat ignite-lab-nodejs

### Tech stack

- NestJS
- Prisma
- Kafka

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```
